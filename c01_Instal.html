1  Instalando o R

1.1 Introdução

Neste tópico, iremos aprender a instalar o R em seu computador. Em geral é indicado que se utilize uma interface para facilitar a utilização da linguagem R, como o RStudio, o VS Code ou diversos outros editores de texto voltados para a programação.

Neste livro, utilizaremos o RStudio, que é um ambiente integrado de desenvolvimento (AID ou do inglês integrated development environment - IDE) que apresenta várias opções e recursos para visualização, marcação de sintaxe e preenchimento automático do script.

1.2 Instalando o R básico (Base-R)

O R é disponibilizado no site www.r-project.org. Ele pode ser instalado em computadores com os sistemas operacionais Windows, Linux ou Mac OSX. Abaixo estão descritos os passos para realizar a instalação (Figura 1.1, Figura 1.2, Figura 1.3).

  1. Acessar o site www.r-project.org;

  2. Entre em CRAN no menu lateral ;

  3. Escolha um espelho (“mirror”), dando preferência aos mais próximos, para realizar o download;

  4. Escolha a versão correspondente ao seu sistema operacional;

  5. Escolha a versão base ;

  6. Faça download da versão mais atual do R disponível;

  7. Abra o instalador R-x.x.x-win.exe (x.x.x representa o número da versão do R), clicando duas vezes nele.

  8. Siga as opções padrão de instalação do R;

  9. Execute o R a partir do menu iniciar. Dependendo do seu sistema operacional você deverá executar a versão de 32 bit ou 64 bit (x32 ou x64);

  10. Ao final do processo você poderá visualizar a interface do R.

1.3 Instalando o Rstudio

Para instalar o RStudio, siga os seguintes passos (Figura 1.4). Atenção: o R-base precisa estar instalado para executar o RStudio!

  1. Entre no website posit.co/download/rstudio-desktop/;
  2. Clique em Install RStudio;
  3. Sigas as instruções para instalação. Caso necessário, na mesma página abaixo, há outras versões do RStudio para serem baixadas.

1.4 O Posit Cloud

O Posit Cloud permite acessar a interface RStudio de um navegador. Para cessar o Posit Cloud (Figura 1.5):

  1. Acesse https://posit.cloud;

  2. Clique em “Get Started”;

  3. Escolha o plano mais adequado (pode ser o gartuito);

  4. Clique em Sign up;

  5. Crie uma conta, podendo ser utilizada a conta Google.
