pinguins <- read_excel("dados_R.xlsx", sheet = "pinguins")

4.1 Medidas de posição e dispersão

Vamos calcular estatísticas descritivas para a coluna correspondente ao comprimento do bico

pinguins$bico_comp #comprimento do bico para cada indivíduo

Utilizaremos as seguintes funções para calcular as estatísticas descritivas:

  • mean(): cálculo da média;

  • median(): cálculo da mediana;

  • var(): cálculo da variância;

  • sd(): cálculo do desvio padrão;

  • summary(): cálculo de diversas estatísticas descritivas.

mean(pinguins$bico_comp) #média
[1] 43.99279
median(pinguins$bico_comp)#mediana
[1] 44.5
var(pinguins$bico_comp) #variância
[1] 29.90633
sd(pinguins$bico_comp) #desvio padrão
[1] 5.468668
summary(pinguins$bico_comp) #mínimo, máximo, mediana, média e 1º e 3º quartis
   Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
  32.10   39.50   44.50   43.99   48.60   59.60

Quantis: podemos utilizar a função quantile() para calcular os quartis ou percentis. Nessa função, o argumento probs pode ser usado para incluir os percentis que vamos calcular. Se não usarmos nenhum valor específico, obteremos aos percentis 0% (menor valor), primeiro quartil (25%), mediana ou segundo quartil (50%), terceiro quartil (75%) e percentil 100% (maior valor):

quantile(pinguins$bico_comp) 
  0%  25%  50%  75% 100% 
32.1 39.5 44.5 48.6 59.6 
# escolher percentis:
quantile(pinguins$bico_comp, probs = c(0.1,0.9))
 10%  90% 
36.6 50.8

4.2 Calculando estatísticas descritivas para múltiplas variáveis ou grupos de valores

Algumas funções podem ser usadas para facilitar a aplicação de funções de forma simultânea em diferentes colunas ou grupos de dados.

4.2.1 Função tapply()

Como temos espécies diferentes, vamos calcular a média e desvio padrão do comprimento do bico para cada uma das espécies utilizando a função tapply().

#média
tapply(pinguins$bico_comp, #coluna com os valores
       pinguins$especie, #coluna com a espécie
       mean) # função a ser aplicada
  Adelia Barbicha   Gentoo 
38.82397 48.83382 47.56807 
#desvio padrão
tapply(pinguins$bico_comp, pinguins$especie, sd) 
  Adelia Barbicha   Gentoo 
2.662597 3.339256 3.106116

4.2.2 Função apply()

Esta função pode ser usada para aplicar outra função para cada coluna de um conjunto de dados. Se quisermos calcular a média em todas as colunas numéricas, podemos calcular em uma por uma, ou utilizar a função apply() para aplicar a função média para todas as colunas

#média de de uma das colunas
apply(pinguins[,3:6], 2, mean) #Calcular a média para todas as colunas
 bico_comp  bico_prof  nadadeira       peso 
  43.99279   17.16486  200.96697 4207.05706

4.2.3 Função aggregate()

Esta função é muito útil quando queremos calcular estatísticas descritivas e temos uma ou mais colunas de fatores que agrupam os dados. Se quisermos calcular as médias para cada uma das variáveis numéricas da planilha pinguins para cada uma das espécies usando aggregate():

aggregate(cbind(bico_comp,bico_prof,nadadeira,peso) ~ especie, #agrupar variaveis numéricas por espécie
          data = pinguins, # selecionar as variáveis
          mean) # função a ser aplicada
   especie bico_comp bico_prof nadadeira     peso
1   Adelia  38.82397  18.34726  190.1027 3706.164
2 Barbicha  48.83382  18.42059  195.8235 3733.088
3   Gentoo  47.56807  14.99664  217.2353 5092.437

Podemos fazer isso também utilizando ambos os fatores, espécie e sexo:

aggregate(cbind(bico_comp, bico_prof, nadadeira, peso)~especie+sexo,data = pinguins, mean) #agregando por espécie e sexo

4.3 Tabelas sofisticadas com table1()

Com o pacote table1, é possível preparar facilmente uma tabela de estatísticas descritivas como a Tabela 4.1 . No exemplo, vamos gerar uma tabela com estatísticas descritivas para as variáveis numéricas do conjunto de dados, calculando-as em separado para cada espécie de pinguim:

library(table1)

Attaching package: 'table1'
The following objects are masked from 'package:base':

    units, units<-
colnames(pinguins) # verificar colunas para escolher
[1] "especie"   "ilha"      "bico_comp" "bico_prof" "nadadeira" "peso"     
[7] "sexo"     
#tabela de variaveis numericas por especie
table1(~ bico_comp + bico_prof + nadadeira + peso | especie , data = pinguins)

Podemos utilizar mais de um fator para separar as espécies, veja o exemplo a seguir como separar as estatísticas descritivas por espécie e por sexo:

#tabela de variaveis numericas por especie e sexo
table1(~ bico_comp + bico_prof + nadadeira + peso|especie + sexo , data = pinguins, overall = NULL)

4.4 Utilizando o tidyverse para calcular estatísticas descritivas

O tidyverse é um conjunto de pacotes versátil para ciência de dados e apresenta ferramentas sofisticadas para calcular estatísticas descritivas (Wickham et al., 2023). Para mais informações, consulte o Capítulo 12 .

4.5 Conclusão

Neste capítulo, exploramos técnicas para calcular e interpretar estatísticas descritivas, essenciais para resumir e compreender conjuntos de dados. Utilizando o conjunto de dados “pinguins”, aprendemos a calcular medidas de posição (média, mediana) e dispersão (variância, desvio padrão), além de identificar quartis e percentis com a função quantile(). Também foram apresentadas funções como tapply(), apply() e aggregate() para calcular estatísticas descritivas de forma agrupada, permitindo análises de forma mais rápida por categorias, como espécie e sexo. Por fim, introduzimos o pacote table1 para criar tabelas sofisticadas de estatísticas descritivas e destacamos a utilidade do tidyverse para análises mais avançadas.
