13.1 Gerando dados aleatórios

O R pode ser utilizado para gerar dados (pseudo) aleatórios. Essas funções são interessantes para realizar simulações de análises

Vamos gerar 100 números aleatórios a partir da distribuição normal, com média 10 e desvio padrão 2

norm10d2 <- rnorm(100, mean = 10, sd = 2)
mean(norm10d2) # média
[1] 10.04513
sd(norm10d2) # desvio padrão
[1] 2.056923

Como são valores aleatórios, a cada vez que forem gerados teremos um valor diferentes. Valores exatamente iguais para reproduzir os resultados podem ser conseguidos executando a função set.seed(x), sendo x un valor qualquer, antes de gerar os dados aleatórios.

Pode-se gerar valores a partir de outras distribuições:

rpois(10, lambda = 10) # gerar 10 valores a partir da distribuição de poisson com lambda = 10
 [1]  7 19  8  4 19  9  9  7  8  7
rbinom(10, 10, 0.5) #distribuição binomial: total de sucessos em 10 rodadas de 10 jogadas de moedas  com 50% de probabilidade de sussesso (sussesso = tirar cara, por exemplo)
 [1] 7 7 6 5 6 4 9 5 5 3

Utilizando valores aleatórios para simular um experimento. no exemplo abaixo (Figura 13.1).

13.2 Fundamentos de programação

13.2.1 Criando funções

As funções são tipos de objeto no R, como podemos observar ao verificar a classe de algumas funções utilizando class():

class(mean) # classe do objeto
[1] "function"
class(lm)
[1] "function"
args(lm) # argumentos da função
function (formula, data, subset, weights, na.action, method = "qr", 
    model = TRUE, x = FALSE, y = FALSE, qr = TRUE, singular.ok = TRUE, 
    contrasts = NULL, offset, ...) 
NULL

Não há uma função no R para calcular o erro padrão, então criaremos uma. Fórmula do erro padrão: \[SE = \sigma/\sqrt N\].

dados <- rnorm(20, mean = 0, sd = 1)
sd(dados)
[1] 1.298958
sd(dados)/sqrt(length(dados)) # Fórmula para calcular o erro padrão
[1] 0.2904559

A partir da fórmula usada para calcular o erro padrão, vamos criar uma função para realizar o cálculo. Podemos criar funções utilizado function(). Vamos criar uma função para calcular o erro padrão. A função terá somente um argumento x, o qual será o vetor sobre o qual será calculado o erro padrão. Neste passo, criaremos uma função chamada ep(). Entre os ‘{}’ incluímos o que a função irá realizar:

ep <- function(x){
  sd(x)/sqrt(length(x))
}

ep(dados)
[1] 0.2904559

Acima criamos a fução ep(x), que realizará o cálculo do erro padrão pela seguinte fórmula: sd(x)/sqrt(length(x).

13.2.2 Condicionais: if() e else()

Os condicionais if e else são usados para controlar o fluxo de execução do código com base em condições específicas.

  • if: Executa um bloco de código se uma condição for verdadeira.

  • else: Pode ser usado em conjunto com if para executar um bloco de código alternativo caso a condição do if seja falsa.

Vamos criar uma função para descrever se um número é maior que 0 ou não. Nesta função, o argumento x será um valor numérico, e a função dirá se esse valor é maior que zero ou não. Desta forma ela seguirá a lógica:

  • Se x > 0 → imprimir(“O valor é maior que zero”)

  • senão → imprimir(“O valor é menor ou igual zero”)

Assim, podemos escrever o código:

maiorquezero <- function(x){
  if (x > 0){print("O valor é maior que zero")}
  else{print("O valor é menor ou igual a zero")}
}

maiorquezero(1)
[1] "O valor é maior que zero"
maiorquezero(-1)
[1] "O valor é menor ou igual a zero"
maiorquezero(0)
[1] "O valor é menor ou igual a zero"

Vamos aperfeiçoar a função para diferenciar valores 0 de valores negativos. Para isso, podemos usar if() 3 vezes:

maiorquezero <- function(x){
  if (x>0){print("O valor é maior que zero")}
  if (x==0){print("O valor é igual a zero")}
  if (x<0){print("O valor é menor que zero")}
}

maiorquezero(1)
[1] "O valor é maior que zero"
maiorquezero(-1)
[1] "O valor é menor que zero"
maiorquezero(0)
[1] "O valor é igual a zero"

Vamos incluir um condicional para aperfeiçoar a nossa função. O condicional permitirá que ela funcione diferente caso desejarmos que ela omita valores faltantes. Desta forma, abaixo nós:

  • Adicionamos o argumento na.rm = FALSE à função. Desta forma, por padrão a função não irá remover valores faltantes, mas se colocarmos na.rm = TRUE, ela executará o condicional no qual ocorrerá a remoção dos valores faltantes.

  • Utilizamos if e else para determinar como a função irá funcionar no caso de optarmos por na.rm = FALSEou na.rm = TRUE. Quando na.rm for TRUE, utilizaremos a função na.omit() para remover os valores faltantes.

ep <- function(x, na.rm = FALSE){
  if(na.rm == FALSE){sd(x)/sqrt(length(x))}
  else{sd(x, na.rm == TRUE)/sqrt(length(na.omit(x)))}
}

ep(dados.na)
[1] NA
ep(dados.na, na.rm = TRUE)
[1] 0.2904559

13.2.3 Loops com for()

A função for() repete um comando por um intervalo de valores. Ela permite executar um bloco de código repetidamente para cada elemento de uma sequência ou vetor. É uma estrutura de controle que permite iterar sobre elementos, aplicando uma operação específica a cada um deles. Vamos utilizá-la para calcular a soma das linhas do conjunto de dados iris (somente as variáveis numéricas).

iris.num <- iris[,1:4] # somente os dados numéricos de iris

  somas <- vector() # cria um vetor numérico para salvar os valores.

for(i in 1:nrow(iris.num)){ # inicia o argumento for - para os valores de 1 ao número de linhas de iris
  somas[i] <- sum(iris.num[i,]) #cria cada valor do vetor iris com a soma dos valores da linha
}

13.2.4 Função return() e print()

A função return( ) pode ser utilizada para retornar o resultado final de um loop. A função print( ) “imprime” os resultados intermediários de cada passo do loop.

mult <- function(x){
  f <- 1
  for(i in 1:x){
    f <- f*i
    print(f)}}

mult2 <- function(x){
  f <- 1
  for(i in 1:x){
    f <- f*i}
  return(f)}

mult3 <- function(x){
  f <- 1
  for(i in 1:x){
    f <- f*i
    return(f)}}

mult(5) # retorna todos os valores gerados a cada iteração
[1] 1
[1] 2
[1] 6
[1] 24
[1] 120
mult2(5) # Retorna somente o último valor
[1] 120
mult3(5) # Retorna somente o primeiro valor, pois foi colocada antes de terminar a iteração
[1] 1
